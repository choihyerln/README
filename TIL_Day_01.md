## CLI
___
> CLI (Command Line Interface, 커맨드 라인 인터페이스)는 터미널을 통해 사용자와 컴퓨터가 상호 작용하는 방식을 뜻한다.

### 터미널 명령어 정리
|명령어|설명|
|:---:|:---:|
|mkdir|폴더 생성|
|touch|파일 생성|
|ls|현재 폴더의 파일 목록 출력|
|cd|다른 폴더로 이동|
|rm|파일 삭제 / 폴더 삭제 (-r옵션)|

### 예시
```bash
$ mkdir test

$ touch a.txt

$ ls
$ ls -a

$ cd ..
$ cd test

$ rm a.txt
$ rm -r test
```
